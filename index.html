   Setting Up an Electronics Lab at Home

Getting Started

Week one is all about getting ourselves set up for success. Please see the topic areas below to make sure that you are prepared for week one and week two.

Click on each topic below for more information.

Kits

We are requiring the purchase of materials for this class which we will be using instead of a textbook. There are two kits that should be ordered directly from Jameco. Be sure to use the discount code "UOC7" for 7% off.

The Parts Kit

This is the the primary kit that we will use for the majority of our labs. You are expected to have this on our first meeting day

The Soldering Kit

On the Jameco kit page you will see the text "You may also be interested in a Solder Practice kit that includes a soldering iron (PN: 2124218)." You will need to purchase this as well.

The soldering kit is separate so that students who already have access to soldering tools will have the option to only purchase the circuit board and not the iron. You may also choose to wait to purchase this kit until week 6 if you want to spread the cost out. This will mean paying more in shipping. Please contact me directly with any questions.

Reading

This week's lab will be all about setting up and getting familiar with your electronics lab at home. To support this activity, I am assigning the reading at Makerspace.com called "Introduction to Basic Electronics" which covers much of the same material we looked at in class. Please read everything up to "How to Determine a Resistor Size". That starts getting into math and then project examples. We will cover that in future weeks.

Fritzing Software

In this class, we will be using an open-source program called Fritzing to design and share circuit layouts. Please create a Fritzing account and download the application so that you can open and manipulate Fritzing files. NOTE: you must create an account in order to download the program without paying a compulsory donation.
